1.「raylibhsp」とは？

「raylibhsp」とは、C/C++言語用のゲーム開発ライブラリの「raylib」をHSP3言語で使えるようにすることを目指したファイル集です。

ライセンスはMIT Licenseとします。

「raylibhsp」には主に以下のファイルが収録されています。

  • 「readme」フォルダ
    この説明書のデータが入ったフォルダです。
  • 「header_maker.hsp」ファイル
    このファイルをHSP3で実行させると「raylib.hsp」ファイルを生成する、HSP3言語スクリプトファイルです。
  • 「data」フォルダ
    「header_maker.hsp」ファイルを実行させるために必要なファイルが入っています。
    • 「callfuncex_module.hsp」ファイル
      32ビット版や64ビット版のHSP3でも同じHSP3言語のスクリプトでdllファイル内の関数を実行できるようにするHSP3言語用モジュールです。このモジュールは単体でも使用できます。
    • 「struct_module.hsp」ファイル
      HSP3言語で構造体のような変数を作成すること等ができるようにするHSP3言語用モジュールです。このモジュールは単体でも使用できます。
    • 「raylib_api.json」ファイル
    • 「raymath_api.json」ファイル
    • 「rlgl_api.json」ファイル
      この3つのファイルは、raylibに同梱されているC言語のプログラムの「raylib_parser.c」をビルドした際に生成される「raylib_parser.exe」実行ファイルを使用して生成されたもので、「raylib.dll」ファイル内の関数情報等が記述されています。
  • 「raylib.hsp」ファイル
    raylibをHSP3言語で使用できるようにする、HSP3言語スクリプトファイルです。前述した「callfuncex_module.hsp」ファイルや「struct_module.hsp」ファイルの中身も、「raylib.hsp」ファイルに内蔵されています。
  • 「hello.hsp」ファイル
    32bit版HSP(UTF-8版)で、raylibのウィンドウを作成し、文字列「Hello, raylibhsp!」と表示させるだけの簡易的なサンプルスクリプトです。このサンプルを実行させるには、「raylib.dll(32bit版)」が必要です。詳しくは、「3.raylibのダウンロード」と「4.「raylib.hsp」を使う前に」を読んでください。
