4.「raylib.hsp」を使う前に

まず、「3.raylibのダウンロード」でダウンロードして解凍したzipファイルのフォルダを開き、中にある「lib」フォルダを開きます。その中にある「raylib.dll」ファイルを、あなたがraylibhspを用いて制作するゲームのフォルダにコピーしてください。同様に、raylibhspのフォルダにある「raylib.hsp」ファイルもコピーしてください。エントリーポイントとなるスクリプトのファイルを「main.hsp」とした場合、制作するゲームのフォルダの中身は以下のようになると思います。

  • 「main.hsp」
  • 「raylib.dll」
  • 「raylib.hsp」
※64ビット版HSP3でゲームを制作する場合は、「raylib.dll」は64ビット版のものを使用してください。そして「2.注意事項」で記述した「hspint64.dll」ファイルと「hspint64.as」ファイルも必要です。
