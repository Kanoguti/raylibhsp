5.「raylib.hsp」を使ってみる

それでは、「raylib.hsp」を実際に使用してみましょう。まず、エントリーポイントとなるHSP3スクリプトファイルを任意のエディタで開いてください。次に、「raylib.hsp」ファイルをインクルードします。

  • 32ビット版の場合
    UTF-8版HSP3を使用することを推奨します。「hsp3utf.as」をインクルードした後に「raylib.hsp」をインクルードしてください。
  • 64ビット版の場合
    64ビット版HSP3を使用するように「hsp3_64.as」をインクルード、それに続いて「HSPInt64.dll」を使用できるように「hspint64.as」をインクルードし、その後に「raylib.hsp」をインクルードしてください。
