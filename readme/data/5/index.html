5.「raylib.hsp」を使ってみる

それでは、「raylib.hsp」を実際に使用してみましょう。まず、エントリーポイントとなるHSP3スクリプトファイルを任意のエディタで開いてください。次に、64ビット版HSP3を使用するように「hsp3_64.as」をインクルード、それに続いて「HSPInt64」を使用できるように「hspint64.as」をインクルードし、その後に「raylib.hsp」をインクルードしてください。
